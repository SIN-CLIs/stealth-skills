# stealth-skills (Private Skill Library)

Plattformspezifische Automatisierungs-Skills für die Stealth Suite.
**Nicht öffentlich** — enthält Betriebsgeheimnisse und rechtlich sensibles Domänenwissen.

## Struktur
```
platforms/heypiggy/   → google-login, survey, modules
platforms/swagbucks/  → (vorbereitet)
_registry.json        → Zentrales Skill-Verzeichnis
```

## Rechtlicher Hinweis
Diese Skills sind ausschließlich für den internen Gebrauch.
Weitergabe kann gegen AGB der Zielplattformen verstoßen.
