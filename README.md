**Nicht öffentlich** — enthält Betriebsgeheimnisse und rechtlich sensibles Domänenwissen.

## Struktur
```
platforms/heypiggy/   → google-login, survey, modules
platforms/swagbucks/  → (vorbereitet)
_registry.json        → Zentrales Skill-Verzeichnis
```

## Rechtlicher Hinweis
